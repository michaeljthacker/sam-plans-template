# Example Instance

**Do not copy this folder in order to use SAM in a project.**

This directory shows an example of `plans/` filled in for a real project.

Do not copy these contents blindly. Use it as a reference for:
- what “good” BUILD/MILESTONE/STATUS look like when filled in
- how thread and state evolve through micro-chunks
## Source project

The example is from [mjt-wordsearch-ui](https://github.com/michaeljthacker/mjt-wordsearch-ui/), a static word search puzzle generator built with plain HTML/CSS/JS. The snapshot here is mid-development — specifically at **B1-M2-P2** (Build 1, Milestone 2, Phase 2), corresponding to commit [`47bab71`](https://github.com/michaeljthacker/mjt-wordsearch-ui/commit/47bab71).

You can use this example in two ways:

1. **Browse here** — read the `plans/` files in this directory to see what a real, in-progress SAM instance looks like.
2. **Follow the repo** — look at the [public repo](https://github.com/michaeljthacker/mjt-wordsearch-ui/) and walk through the commit history from scratch to completed B1, watching the project and its `plans/` folder develop commit by commit.
