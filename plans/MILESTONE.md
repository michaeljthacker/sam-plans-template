# MILESTONE — Template

## Purpose
`MILESTONE.md` describes the **multi-phase implementation plan for the current Milestone**:
- phases, steps, acceptance criteria,
- relevant technical approach for the milestone,
- any milestone-specific constraints.

A Milestone is one step in the implementation plan for a Build.

## When to update
- When switching to a new active milestone (per branch)
- When phase definitions change
- When acceptance criteria change

## Required sections
- Milestone ID and goal
- Phases (each with acceptance criteria)
- Notes / dependencies (optional)
